     1、一级企业操作转移：一级企业对下级、一级企业对生态合作。

     + 状态：全部、待审核（企业端=审核中）、审核通过(企业端=转移成功)、审核未通过(企业端=转移失败)

申请时间

列表全字段：

     对于状态、申请时间、审核时间的排序是基于整个列表的，并不是当前页面！

     1、申请企业：企业名称 + ID

     2、转移设备：转移设备的数量，点击可查看详情即转移详情，原型所示

     3、状态：

          1、待审核：对应企业端 = 审核中，发起申请转移后的默认状态

          2、审核通过：对应企业端=转移成功

          3、审核未通过：对应企业端=转移失败

      4、转入企业：企业名称 + ID

      5、转移类型：PCBA设备/整机设备，发起转移是通过PCBA转移的展示PCBA设备，整机设备同理。

      6、授权产品：产品名称 + ID

      7、申请时间：转移发起时间

      8、审核时间：管理员审核通过的时间

      9、转移说明：企业填写的申请转移说明


发起审核：

1、审核通过：

         设备所属权转移，会进入接收方，接收方可对设备 分配销售、绑定App、设备转移、OTA升级 操作。

         注意处理以下细节：

         1、分成处理

                 若License 转移前存在已绑定App，若绑定的为牵心公版App。鹤梦平台需自动给接收方企业分成（按照平台对接收企业级分成比例，不存在即不分成）。

         2、产品权限

                 发起方（企业）的授权整机产品以及对应的固件需要对接收方全部可见（属动态关联，非快照），接收方只读，对已转移的设备只有发布固件OTA“写”权限。

         3、若设备转移存在 A -> B -> C，则C企业对A企业的产品同上述，可见。


2、审核未通过：需填写驳回原因，展示给企业发起方——申请方。
